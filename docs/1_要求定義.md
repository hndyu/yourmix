# 要求定義
プロジェクトの目的と全体像を定義する。

## プロジェクト概要
本プロジェクトは、「YourMix」という名前のカクテル共有プラットフォームの制作を目的としています。

YourMixは、ユーザーがカジュアルにオリジナルカクテルを創造したり、アレンジしたり、共有したりできるコミュニティサイトです。

## コンセプト
- 専門知識がなくても、気軽にカクテル作りを楽しめるプラットフォーム
- 「とりあえず作ってみたくなる」手軽でポジティブなコミュニティ
- 子どもがドリンクバーでジュースを混ぜる感覚で、大人も気軽に試したくなる場所
- カクテル作りの敷居を下げ、楽しさを共有できる空間
- アルコールが苦手でもノンアルカクテルで楽しさを共有できる体験

## 目的
- カクテル作りの楽しさを広める
- ユーザー同士のレシピ共有によるコミュニティ形成
- カクテル作りの知識やテクニックの普及
- 手元にある材料で作れるカクテルを簡単に見つけられる環境の提供
- AIを活用した新しいカクテル体験の創出

## 対象ユーザー
- カクテルに興味がある20〜40代
- 家飲みを楽しみたい人
- ホームパーティーを開催する人
- お酒の種類や組み合わせを探求したい人
- SNSでドリンク写真を共有するのが好きな人
- バーテンダーや飲食業界従事者

## ビジネス目標
- 月間アクティブユーザー
- ユーザーエンゲージメント（レシピ投稿・レシピ評価）
- アフィリエイト収入の確保
- 将来的な有料会員プランの導入
- アルコール・飲料メーカーとのタイアップ実現

## 主要機能要求
1. カクテルのレシピ検索機能[MVP]
2. 手元にある酒・割材の登録・削除機能
3. 登録した材料で作れるカクテルの検索・絞り込み機能
4. レシピに対する不足材料の表示機能
5. 「今日の一杯」：日替わりのおすすめレシピ表示機能[MVP]
6. LLMによるオリジナルカクテル生成機能[MVP]
7. 会員登録・ソーシャルログイン
8. 既存レシピから派生したアレンジカクテルの投稿機能
9. 酒・割材のアフィリエイトリンク機能
10. 「おいしい」ボタン（いいね機能）
11. レシピ保存機能
12. 広告表示機能

## MVP
1. カクテルのレシピ検索機能
2. 「今日の一杯」：日替わりのおすすめレシピ表示機能
3. LLMによるオリジナルカクテル生成機能

## ビジュアルアイデンティティ（Visual Identity）
### ロゴ・ロゴタイプ
- 名称の親しみやすさを活かし、手書き風やカクテルの液体をイメージした流れるようなフォント
- シンボル案：シェイカー・グラス・ドロップ・星（「ひらめき」「実験的」なニュアンス）

### カラーパレット
| 色 | 役割 | 備考 |
| ---- | ---- | ---- |
| メイン：ピーチピンク / トロピカルオレンジ | フレンドリー・陽気・遊び心 | カクテルの色を想起させる暖色系 |
| アクセント：ターコイズ / ミントグリーン | 爽快感・クールさ | ノンアル感やバランス感を演出 |
| ベース：オフホワイト / ライトグレー | 清潔感・背景色 | レシピ写真を引き立てる |

### 写真・イラストのトーン
- 写真：ライトトーンで明るく淡い
- 装飾イラスト：パステルカラーの手描き風（液体・泡・星・シェイカーなど）

## 制約条件
- レスポンシブデザイン対応（PC・タブレット・スマートフォン）
- PWA対応
- アクセシビリティ対応（WCAG 2.1 AAレベル）
- アルコール関連コンテンツに関する法的規制への対応
- 適切な飲酒を促進するためのガードレールの必要性
- 未成年者への配慮（年齢確認機能の実装）
- プライバシーの尊重と法律の遵守
- 個人開発のため予算はできる限り少なくする

## 優先度
1. ユーザビリティ（直感的で使いやすいUI/UX）
2. コミュニティ機能（共有・交流の活性化）
3. パフォーマンス
4. セキュリティ

## ステークホルダー
- プロジェクトオーナー（ほか全て兼務・1名）
- 一般ユーザー
- 飲料メーカー（潜在的パートナー）
- バーテンダー（潜在的アドバイザー・インフルエンサー）
