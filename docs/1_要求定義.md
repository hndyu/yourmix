# 要求定義

プロジェクトの目的と全体像を定義する。

## 1. プロジェクト概要

本プロジェクトは、「YourMix」という名前のカクテル共有プラットフォームの制作を目的としている。

YourMixは、ユーザーが手元の材料をもとに既存カクテルレシピを発見し、またはLLM（大規模言語モデル）を活用してオリジナルカクテルを創造・共有できるコミュニティサイトである。

YourMixはプロジェクトオーナーのエンジニアとしての転職に向け、ポートフォリオに記載するために制作されるが、単体のサイトとして成り立たせる。

## 2. コンセプト

- 専門知識がなくても、気軽にカクテル作りを楽しめるプラットフォーム  
- 「とりあえず作ってみたくなる」手軽でポジティブなコミュニティ  
- 子どもがドリンクバーでジュースを混ぜる感覚で、大人も気軽にカクテル作りを試したくなる、新しい発見と楽しさを提供する場所  
- カクテル作りの敷居を下げ、楽しさを共有できる空間  
- アルコールが苦手でもノンアルカクテルで楽しさを共有できる体験

## 3. 目的

- カクテル作りの楽しさを広める  
- ユーザー同士のレシピ共有によるコミュニティ形成  
- カクテル作りの知識やテクニックの普及  
- 手元の材料から、既存レシピの発見とAIによるオリジナルカクテルの生成を組み合わせた、新しいカクテル体験の提供

## 4. 対象ユーザー

- カクテルに興味がある20〜40代  
- 家飲みを楽しみたい人  
- ホームパーティーを開催する人  
- お酒の種類や組み合わせを探求したい人  
- SNSでドリンク写真を共有するのが好きな人  
- バーテンダーや飲食業界従事者

## 5. ビジネス目標

- 月間アクティブユーザー  
- ユーザーエンゲージメント（レシピ投稿・レシピ評価）  
- アフィリエイト収入の確保  
- 将来的な有料会員プランの導入  
- アルコール・飲料メーカーとのタイアップ実現

## 6. 主要機能要求

- 材料投入型レシピ検索・生成機能 (Mixボタン) \[MVP\]  
  - ユーザーが入力した材料から、まず既存カクテルデータベースを検索し、該当レシピがあればその作り方を表示する  
  - データベースに該当レシピが存在しない場合、LLM（大規模言語モデル）を活用してオリジナルカクテルレシピを自動生成し提案する  
    - また、ユーザーが入力した材料を含む近似のカクテルをデータベースから抽出し、候補としてリスト表示する  
    - また、オリジナルレシピをコミュニティに共有できる  
  - この機能は、「Mixボタン」として提供され、簡単なアニメーションを伴い、ユーザーに直感的で楽しい体験を提供する。  
- 日替わりのおすすめレシピ表示機能  
- レシピのSNS共有機能  
- 酒・割材のアフィリエイトリンク機能  
- 会員登録・ソーシャルログイン  
  - 手元にある酒・割材の登録・削除機能  
  - 登録した材料で作れるカクテルの検索・絞り込み機能  
  - レシピに対する不足材料の表示機能  
  - 「おいしい」ボタン（いいね機能）  
  - レシピお気に入り機能  
  - カクテル閲覧履歴機能  
- 広告表示機能

## 7. 制約条件

- レスポンシブデザイン対応（PC・タブレット・スマートフォン）  
- PWA対応  
- アクセシビリティ対応（WCAG 2.1 AAレベル）  
- アルコール関連コンテンツに関する法的規制への対応  
- 適切な飲酒を促進するためのガードレールの必要性  
- 未成年者への配慮（年齢確認機能の実装）  
- プライバシーの尊重と法律の遵守  
- 個人開発のため予算はできる限りゼロを貫く

## 8\. 優先度

1. コミュニティ機能（共有・交流の活性化）  
2. ユーザビリティ（直感的で使いやすいUI/UX）  
3. パフォーマンス  
4. セキュリティ

## 9\. ステークホルダー

- プロジェクトオーナー（個人プロジェクトのため、デザイナーやプログラマーなどすべて兼務・1名）  
- エンドユーザー  
- 飲料メーカー（潜在的パートナー）  
- バーテンダー（潜在的アドバイザー・インフルエンサー）
